2.Superfluous Words

本人上网经历不长，高中毕业以后才开始知道Q-zone，一路还玩过一阵，如今基本已经荒废。果然是人老了，跟不上了。

大学二年级才开始转网页，那个时候博客-中国还很火的样子，带动了不少人的写作热情，正是一段无聊的学生岁月，也就跟着开始一本正经的码字，完全是自娱自乐，好不快活。说实话，那个刚刚兴起的事物，你完全不用考虑它的技术性或者自己的内在需求，玩一段时间，自然就疲软了。更加由于学习、工作的压力，你需要有一个半私密半专业性质的平台，提供给你整理知识之用。

后来，由于不知道何故，又私下转到网易上去了，一是安静，认识的少，自己写自己看，不为其他；二网易的风格相对对胃口，看新闻，转载，邮件，都很方便。当然，由于算半个码农的缘故，博客园、豆瓣这些社区论坛，大多都有一些散落的零星软文。一直疏于整理和耕耘。

常在网上转悠，受益于别人的东西良多，内心未免亏欠。尤其是看到那些大牛的个人网站、个人博客之类的，甚是拜服。一直也想自己做一个。以前有想过wordpress之类的，但是又是数据库有事php的，配置真是一个折腾。偶然机会看到jekyll，就一路按按关注，发现极对胃口。自己定制，写作灵活，管理方便，备份容易。。。于是，动手，开始干起来了。

经过这几天的费心折腾，本人的 个人技术Blog 终于上线啦！

就像自己用尽时间和心力搭建了一个小狗窝，别人再如何如何不满意，自己内心也是有几分热乎乎的。
其中寓意，在沙滩上搭积木的人不言自明。

虽然暂时搭在Github上，有点小打小闹，占便宜的嫌疑，但是既然来了，当然也就本着
互联网开源和分享的精神，以后一定努力 coding，希望也你能为社区做贡献。

3.Hello,Octopress!

为了搭建这个博客，我也是一路疯狂 google 和不断的 git 和 rake 啊！连续两周周末，腰酸背疼，有么有！

话说很多人的 first blog 大概也是讲述自己一路如何如何辛苦，如何如何一步步定制和实施，终于守得云开，豁然开朗的
感觉。本人自认为不是技术达人，也不追赶潮流，完全是在各位攻略和路线图指导下下来的，中间 ruby 安装和本地 web 配置费劲了一把，反正是阴差阳错就成了现在这个样子了。暂时受用，由于没有做 web 网页的经验，一路都是跛脚的 ruby，html，markdown 等瞎掺和，也蛮有趣的。

如果不懂安装的童鞋，请自行谷歌Octoprsse，很多达人分享经验，也可以直接去官网上看。
难倒不是太难，ruby 以后 git，gem，rake 等就可以了。最最可怖的就是 ruby 的安装，本人被掠了几十遍不止，都没脾气了。所以，如果真心想要体验 octopress 的惊艳和魅力，
遇到困难呢，一定不要放弃，困难是暂时的，最后你一定你能找到方法的。Come on！开始你的octpress之旅吧。

推荐一个大牛的博客 TheJoyOfHack（请果断Google之，大神作品，你值得一看！)。

无论是页面布局还是字体色彩，真是艺术品啊! 比起国内网站牛皮廯一样的爬虫广告，这个页面真是清爽简洁又生动活泼、干净利落又丰富多彩、科学潮流又技术魅力。这才是真正的个人博客啊！
你看，你原来可以做得这么好！那就开始吧！我试过很多主题，最后还是决定按这个来，模仿也是一种崇拜吧！
